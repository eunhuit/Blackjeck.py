## 구현 기능

### 기능 1
      openpyxl 라이브러리로 데이터를 액셀화 시킴
### 기능 2
      계좌(돈) 개념으로 좀더 재미있는 진행
### 기능 3
      자동합산 기능으로 편리한 게임
### 기능 4
      대문자 단축어 기능 h, s 와같은 단축어로 빠른 진행 가능!
